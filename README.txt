site2 structure

- index.html
- content/index.yaml
- src/components/*.js
- src/styles/theme.css
- assets/*.png

how to edit

- change copy in content/index.yaml
- change look in src/styles/theme.css
- change section markup in src/components/

run

- open with local server
- example: python3 -m http.server
- then open /site2/
